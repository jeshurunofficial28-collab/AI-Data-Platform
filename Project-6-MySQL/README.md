# 🚀 Project 6 - MySQL Business Analysis

## 📌 Project Overview

This project demonstrates how to use MySQL for storing, querying, and analyzing business datasets. Five different datasets were imported into MySQL, and SQL was used to answer real business questions and generate meaningful insights.

---

## 🎯 Objectives

- Create and manage a MySQL database.
- Import CSV datasets into MySQL.
- Write SQL queries for data analysis.
- Perform business analysis using SQL.
- Prepare the data for the next Python ETL project.

## 📊 Business Analysis Performed

### Customers
- Customer distribution by gender
- Top states by customer count
- Average customer age
- Preferred registration channel

### Products
- Product distribution by category
- Most expensive products
- Average price by category
- Top brands

### Orders
- Orders by currency
- Average quantity per currency
- Highest quantity orders
- Daily sales analysis

### Shipment
- Shipment distribution by country
- Shipment mode analysis
- Top vendors
- Freight cost analysis

### Amazon Delivery
- Weather vs delivery time
- Traffic vs delivery time
- Vehicle performance
- Area-wise delivery analysis

---

## 💡 Key Business Insights

- California has the highest number of customers.
- Home Appliances is the largest product category.
- USD has the highest number of orders.
- Air is the most frequently used shipment mode.
- Cloudy weather and traffic jams increase delivery time.
