# 📊 Project 8 – Data Visualization with Python

## 📌 Project Overview

This project demonstrates how to create professional business visualizations using Python, Pandas, and Matplotlib. Data is extracted from a MySQL database and transformed into meaningful charts that help analyze customer behavior, product pricing, order trends, and delivery performance.

---

## 🛠 Technologies Used

- Python
- Pandas
- Matplotlib
- MySQL
- MySQL Connector

## 📊 Visualizations Created

### 1. Customer Gender Distribution (Bar Chart)

- Compares the number of customers by gender.

### 2. Customer Gender Distribution (Pie Chart)

- Displays the percentage distribution of customer genders.

### 3. Orders Over Time (Line Chart)

- Shows order trends over time.

### 4. Delivery Time Distribution (Histogram)

- Analyzes delivery time frequency.

### 5. Unit Cost vs Unit Price (Scatter Plot)

- Examines the relationship between product cost and selling price.

### 6. Business Dashboard

A dashboard combining multiple charts into a single business report.
