# 🚀 Project 7 - Python ETL Pipeline

## 📌 Project Overview

This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline using Python, MySQL, and Pandas.

The project connects Python to a MySQL database, extracts multiple datasets, performs business analysis, and exports summary reports as CSV files.

---

## 🎯 Objectives

- Connect Python to MySQL
- Extract data from database tables
- Load data into Pandas DataFrames
- Perform exploratory data analysis
- Generate business insights
- Export summary reports
