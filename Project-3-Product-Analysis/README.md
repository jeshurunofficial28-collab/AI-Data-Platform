# 📦 Product Analysis using Python

## 📌 Project Overview

This project analyzes a product dataset from a global electronics retailer using Python and Pandas.

The project covers:

- Reading the dataset
- Exploring the data
- Checking missing values
- Generating statistics
- Cleaning data
- Creating visualizations
- Producing business insights

---

## 🛠️ Tools Used

- Python
- Pandas
- Matplotlib
- VS Code
